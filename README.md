# gmin

A minimalist biologically plausible architecture geared towards a unified implementation of processing in mid-level vision.

**Try it out: [gmin.klab.lt](http://gmin.klab.lt)**

# Dependencies

- Python 2.7
- numpy
- scipy
- matplotlib
- sklearn
